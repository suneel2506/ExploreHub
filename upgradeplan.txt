## PROJECT GOAL

ExploreHub currently contains more than 124,000 OpenStreetMap places.

The existing enrichment system works only for famous places because it depends mainly on Wikipedia and Wikidata.

This is NOT enough.

I want EVERY place in the database to have meaningful information.

The objective is to create a Universal Multi-Tier Enrichment System.
